Typeahead Multiselect

An enhanced multiselect with typeahead and scrolling. This library can be used to build more complex examples, see angular-ui/ui-select2 for more info.

Dependencies select2 angular-ui-select2 jquery

Example

Usage

Setting Width

It is recommended to setup input elements to have 100% column width and that's the default in this case. However, if you want specify a different width you can do so by passing it as an option: ui-select2="{ width: '50%' }". It accepts both px and percent values.

Includes

Make sure you include the following files in index.html and inject 'ui.select2' in your app angular.module('styleGuideApp', ['ui.select2']);

bower_components/jquery/jquery.js bower_components/select2/select2.js bower_components/angular-ui-select2/src/select2.js
