UI Tooltip

A lightweight, extensible directive for fancy tooltip creation. The tooltip directive supports multiple placements, optional transition animation, and more. For use in RiseVision, the tooltip trigger should be `click`. This library can be used to build more complex examples, see UI Bootstrap for more info.

Dependencies bootstrap angular

Script Includes

Make sure you include the following files in index.html

bower_components/bootstrap/bootstrap.css bower_components/angular/angular.js bower_components/angular-bootstrap/tooltip/tooltip.js
