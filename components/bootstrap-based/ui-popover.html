UI Popover

A lightweight, extensible directive for fancy popover creation. The popover directive supports multiple placements, optional transition animation, and more. For Risevision, the trigger should be set to 'click' so that it will work on mobile devices. This library can be used to build more complex examples, see UI Popover for more info.

Dependencies bootstrap CSS angular UI Bootstrap

Module Includes

Declare a dependency on the ui. in your module:

angular.module('myApp', ['ui.bootstrap']);

Script Includes

Make sure you include the following files in index.html

bower_components/angular/angular.js bower_components/angular-bootstrap/ui-bootstrap-tpls.js
