Google Drive Picker

An Angular directive that provides a button to launch the Google Drive Picker modal and select a file based on a file type that you provide.

Dependencies Google Client JSAPI

Usage

When clicking the Google Drive Picker button, you may need to authenticate yourself to use Google Picker by entering your Google credentials. Once authorized, the Google Picker modal is launched and you are prompted to select a file of your choice. By default, all Google Drive document types are displayed. You can specify a particular document type by providing a view-id attribute to the directive and the appropriate view type as the value. Refer to the Google Picker Reference Guide for a complete list of View Ids.

An event will be $emit called cancel if you click the "Cancel" button in the Google Picker modal.

An event will be $emit called picked when you select a file in the Google Picker modal and this event will provide the Google Response.DOCUMENTS array as its data. Refer to the Google Picker Reference Guide for more details regarding the structure of the data.
